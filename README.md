# carrito-compras-vue3

Un carrito de compras usando Vue 3 y Bootstrap v4.6.

[DEMO](https://carrito-compras-vue3-dan.netlify.app/)

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```
